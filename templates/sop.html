Standard Operating Procedures

Operational guidelines and help documentation.

1. Introduction to EMCITOne

The EMCITOne Portal is the central nervous system for Ecometrix's IT infrastructure. It replaces manual spreadsheets with a unified, automated platform for tracking assets, managing support requests, and ensuring compliance.

Core Objective
To maintain 99.9% uptime of IT assets and resolve support tickets within the defined SLA timelines.

2. Access & Security

Login Procedure
  • Use only the secure internal URL provided by IT.
  • Login with your official @ecometrix.co.in email.
  • First Time Login: Use the temporary password provided in your onboarding email.
Roles & Permissions
  • Super Admin Full system control, Vault access, Data Export.
  • Admin Asset & Ticket management. Read-only Vault.
  • User Raise tickets, View own assets.

3. Ticket Management Workflow

Step 1: Raising a Ticket

Navigate to Dashboard > New Ticket. Select category (Hardware/Software) and attach a screenshot if applicable.

Step 2: Admin Triage

IT Admin reviews the ticket. Status changes to In-Progress. If hardware replacement is needed, approval is requested.

Step 3: Resolution & Closure

Once resolved, Admin adds notes and closes the ticket. User receives an email notification.

4. Asset Lifecycle

Every piece of hardware follows a strict lifecycle from procurement to disposal.

Stage Action Required
Procurement Add to "Asset Manager". Fill Section A (Details) & Section B (Invoice/Warranty).
Deployment Update "Assigned To" field. Status automatically changes to In Use.
Maintenance Log issues in "Repair Logs". Status changes to Under Repair.
Disposal Mark as Scrap. Archive data before physical disposal.

5. User Management

Admins are responsible for onboarding and offboarding employees in the system.

Onboarding

Create user profile. Map to correct Department. Default password is Welcome@2026.

Offboarding

Collect all assigned assets. Mark assets as "In Stock". Deactivate user account.

6. Security Protocols

Secure Vault

All software keys must be stored in the Vault. Access is logged.

Captcha Protection

Sensitive data reveal requires solving a mathematical challenge.

Audit Trails

Every creation, deletion, and edit is timestamped in the database.
